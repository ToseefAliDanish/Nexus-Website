# 🚀 Business Nexus Platform

## 📌 Overview

This project is a **frontend prototype** of the *Business Nexus Platform*, designed to connect investors and entrepreneurs.
It demonstrates key features like scheduling, communication, document handling, payments, and security — all implemented using simple web technologies.

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## ✨ Features

### 📅 Meeting Calendar

* View meeting date
* Accept / Decline meeting

### 📹 Video Call (UI)

* Start / End call buttons
* Call status display

### 📄 Document Chamber

* Upload documents
* Change document status (Draft / Review / Signed)
* Add signature

### 💳 Payment System (Mock)

* Wallet balance display
* Deposit / Withdraw functionality
* Transaction history

### 🔐 Login & Security

* Login form with Email & Password
* OTP input field
* Basic validation

---

## 📱 Responsive Design

* Works on desktop and mobile devices
* Adaptive layout using CSS

---

## ▶️ How to Run

1. Download or clone the repository
2. Open the folder
3. Double-click `index.html`

👉 No installation or setup required
